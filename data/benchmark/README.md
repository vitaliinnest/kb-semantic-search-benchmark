# Шаблон бенчмарку

Ця папка містить шаблон бенчмарку для семантичного пошуку.
Вона включає опис корпусу та gold-standard відповідностей запитів до релевантних чанків/документів.

## Файли

- sources.md: опис корпусу (джерела, тематики, мови)
- queries.jsonl: список запитів із метаданими
- qrels.jsonl: gold-standard релевантності (запит -> релевантні chunk_id/doc_id)

## Як використовувати

1) Заповніть sources.md реальним описом корпусу.
2) Заповніть queries.jsonl реальними запитами.
3) Заповніть qrels.jsonl оцінками релевантності, що посилаються на chunk_id або doc_id.

## Нотатки

- Використовуйте chunk_id, згенеровані ingest_chunk.py (формат: "relative/path:chunk_index").
- Якщо потрібна релевантність на рівні документа, задайте doc_id, а chunk_id залиште порожнім.
- Оцінки релевантності мають бути цілими: 1 = релевантний, 2 = дуже релевантний.

# Медичний корпус — довідка

## Джерела

- `mock_anatomy.md` — Витяги з підручника з анатомії та фізіології людини (мок-дані). Охоплює серцево-судинну, дихальну, нервову, травну та опорно-рухову системи.
- `mock_pharmacology.md` — Витяги з підручника з фармакології (мок-дані). Охоплює загальну фармакологію (фармакодинаміка, фармакокінетика), антибіотики, серцево-судинні препарати, аналгетики та НПЗЗ.

## Бенчмарк

- `queries.jsonl` — 20 запитів (M001–M020) українською мовою з медичної тематики.
- `qrels.jsonl` — Оцінки релевантності (doc-level: `chunk_id: null`, `relevance: 2`).
- Типи запитів: `definition`, `factual`, `procedural`, `classification`, `comparison`, `technical`.

## Формат

- `chunk_id` присутній → матч лише на рівні чанку.
- `chunk_id: null` → матч на рівні документа через поле `source`.
- `relevance`: 1 = релевантний, 2 = дуже релевантний.

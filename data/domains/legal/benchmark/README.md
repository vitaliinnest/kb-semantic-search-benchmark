# Шаблон бенчмарку — Юридична база знань

Ця папка містить бенчмарк для оцінки семантичного пошуку по юридичному корпусу.

## Файли

- sources.md: опис корпусу та джерел
- queries.jsonl: 20 тестових запитів (визначення, факти, порівняння)
- qrels.jsonl: gold-standard релевантності (doc-level)

## Типи запитів

- `definition` — запити на визначення правових понять
- `factual` — запити на конкретні факти, норми, строки
- `procedural` — запити на порядок дій або вимоги
- `comparison` — запити на порівняння схожих понять
- `consequence` — запити на правові наслідки

## Як використовувати

1. Завантажте повні тексти кодексів у `raw/` (замість mock-файлів).
2. Запустіть чанкінг: `Файли → Чанкінг` (домен: Юридична база).
3. Побудуйте індекси: `Індекси` (домен: Юридична база).
4. Запустіть benchmark: `Benchmark` (домен: Юридична база).

## Нотатки

- Офіційні тексти кодексів: https://zakon.rada.gov.ua/
- Для chunk-level qrels замініть doc_id на конкретний chunk_id після чанкінгу.
- Рекомендований top_k=10 для валідного порівняння з tech-доменом.
